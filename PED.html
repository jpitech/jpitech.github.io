Card Reader Troubleshooting

Re-pair and open the card reader manually.

  1. Go to Cashless Maintenance Menu
  2. Go to Manager → Support → Support
  3. Password for second support menu is “Last Digit Year, Last Digit Month, Last Digit Day.” Password for 10/15/2020 would be “050”
  4. Tap “Cashless Maint”
  5. Tap “Pair PED” and type the code shown into the card reader. The code is NOT case-sensitive.
  6. Tap “Open PED”
