Card Reader Troubleshooting

Re-pair and open the card reader manually.

Access the Cashless Maintenance menu.

  1. Go to Manager → Support → Support
  2. The password is test.
  3. Tap “Cashless Maint”
  4. Tap “Pair PED” and type the code shown into the card reader. The code is NOT case-sensitive.
  5. Tap “Open PED”
